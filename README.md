# ArteMatch — Landing Page (Entrega del challenge)

## Información del Proyecto
ArteMatch es una plataforma para conectar artistas independientes con clientes, diseñada para ofrecer visibilidad profesional, formalizar encargos y facilitar la contratación de servicios artísticos.  
El proyecto completo contempla una plataforma web y aplicaciones móviles; **en este challenge se solicita la entrega de la Landing Page**, que presenta la propuesta de valor, secciones informativas y llamadas a la acción para atraer usuarios a la plataforma principal.

## Segmentos Objetivo
- Artistas independientes que buscan visibilidad y formalizar encargos.  
- Usuarios interesados en descubrir y contratar talento local.

## Principales características ofrecidas por la solución
- Presentación clara del proyecto y propuesta de valor.  
- Explicación “Cómo funciona” la contratación.  
- Razones para elegir el servicio frente a alternativas.  
- Beneficios clave para artistas y clientes.  
- Testimonios o reseñas para generar confianza.  
- Sección de preguntas frecuentes (FAQ).  
- Botones de registro o contacto (CTA).  
- Diseño responsive basado en principios de IHC.
